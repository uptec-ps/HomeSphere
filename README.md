# HomeSphere
HomeSphere ist ein schlankes, benutzerfreundliches Verwaltungs-OS, verpackt in einem Docker-Container. Es wurde entwickelt, um die Überwachung und Konfiguration Ihres Ubuntu Heimservers so einfach wie möglich zu machen – ganz ohne komplexe Kommandozeilen. Ihr gesamtes Heimnetzwerk, zentral und sicher in einer Sphäre verwaltet.
